# Student Course Management System

## Overview
This is a C++ console application that allows managing students and their enrolled courses.  
Users can add, remove, search, and display students, as well as manage course enrollments.

---

## Features
- Add Student (ID, Name, GPA)
- Remove Student by ID
- Search Student by ID
- Display All Students
- Enroll Student in Course
- Show Student Courses
- Sort Students by GPA
- Menu-driven interface

---

## Files
- `Person.h` → Base class for Person
- `Student.h` / `Student.cpp` → Derived class for Student with course management
- `main.cpp` → Program entry point, menu and logic
- `UML class diagram of inheritance` → UML diagram for the system
- `report.pdf` → Detailed project report

---

## Requirements
- C++ Compiler (Visual Studio, GCC, etc.)
- Standard C++ Library (STL)

---

## How to Compile and Run

### Using Visual Studio
1. Open Visual Studio.
2. Create a new C++ Console Project.
3. Add all files from `src/` folder to the project.
4. Build the solution (Ctrl+Shift+B).
5. Run the project (F5).
